⚙️ Como integrar com a Kiwify (automático)

Como o Vercel é estático, a geração automática de chaves após pagamento funciona assim:


Opção 1 — Manual (funciona agora):

1. Cliente paga na Kiwify

2. Kiwify te envia e-mail com dados do comprador

3. Você abre este painel, gera a chave e copia o link

4. Responde o e-mail do cliente com o link de acesso


Opção 2 — Automático via Kiwify (configuração extra):

Na Kiwify: Produto → Entrega → "Entrega por URL externa"

Cole este link na entrega:

seusite.vercel.app/app?key=CHAVE-GERADA

Crie uma chave genérica PRO e use ela no link de entrega da Kiwify. Todos os clientes receberão essa mesma URL no e-mail da Kiwify automaticamente.
